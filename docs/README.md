## Key facts for AI agents

- **No ORM, no sqlc** — all queries are raw `pgx`. Read handler or service code to find the SQL.
- **FIFO cost basis is computed in Go** (`portfolio_calculator.go:computeFIFOPositions`), not SQL. Backfill snapshots use a simpler avg-cost formula — this causes live P/L vs Trends P/L divergence.
- **Auth = HttpOnly JWT cookie** (`auth_token`). All `/api/*` routes except `/api/auth/login`, `/api/auth/setup`, `/api/auth/status` require it.
- **AI features** (Signal analysis, metals drivers) read `ai_provider`/`ai_api_key`/`ai_model` from `app_settings` table. Configure via Settings → Signal UI.
- **TimescaleDB** — `prices`, `portfolio_snapshots`, and `market_data` are hypertables. Use `time_bucket()` and `last()` aggregates for time-series queries.
- **Frontend build requires manual copy**: `cd web && pnpm build` → then `cp -r web/dist internal/web/dist` (or `make build` does both).
- **Market Mood data**: `market_data` table is populated by `SyncMarketData` (niftyindices.com POST API). MMI comes from `api.tickertape.in/mmi/now` with `Origin: https://www.tickertape.in` header. Precious metals from Yahoo Finance `GC=F`, `SI=F`, `USDINR=X`.
- **routeTree.gen.ts** is auto-generated by TanStack Router Vite plugin — never edit manually.

## Common debugging entry points

| Symptom | Start here |
|---------|-----------|
| Dashboard P/L ≠ Trends P/L | `06-integrations-debugging.md` → FIFO vs avg-cost divergence |
| Market Mood page empty | `06-integrations-debugging.md` → MMI / market_data section |
| AI suggestions not appearing | `app_settings` table has no `ai_api_key`; configure at `/settings` |
| Holdings invested amount wrong | BONUS unit handling in `portfolio_calculator.go:computeFIFOPositions` |
| Import parse fails | `internal/parsers/detector.go` + broker-specific parser |
| New endpoint needed | Add route in `router.go`, handler file, type in `web/src/lib/api.ts` |
| New DB column needed | Add goose migration in `internal/db/migrations/` with next sequence number |
| Price not updating | `internal/jobs/scheduler.go` price fetch job + `price_fetcher.go` |
